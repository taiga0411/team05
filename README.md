アプリの概要
<img width="792" height="591" alt="image" src="https://github.com/user-attachments/assets/b4800ca1-a34f-4935-b412-b6d6d24c1b90" />

-	自キャラは画面手前で、武器を持って背中を向けており、左右移動のみ可能
-	敵キャラは画面奥から集団でこちらに向かってゆっくり向かってくる
-	武器を使って敵を倒す
-	定期的に、触れると自キャラの数が増減する壁が敵キャラとともに出現する。この壁を攻撃すると壁の数字が増減し、数字に応じて自キャラの数が変化する
-	自キャラが敵、もしくは自分より数の大きい赤いゲートに触れるとゲームオーバー
